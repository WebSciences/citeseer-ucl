Provides code to augment the Heritrix crawler for use with the CiteSeerX ingestion system. This includes JMS bridges and notification modules for submitting new jobs to Heritrix via JMS and subsequently monitoring their progress. Filters and extra processors are included that allow downloading of targeted content and writing additional metadata regarding downloaded materials.
